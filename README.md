# AlgorithmsJS

Some Algorithms written in Bash for language studies (PR are welcome)

## Requirements

- Bash ^4.4.5
- (Bats)[https://github.com/sstephenson/bats]

## Algorithms

| Name 						          | Solutions 								            | Description 										                  |
|---------------------------|---------------------------------------|---------------------------------------------------|
| ReverseString			        | 1 								                    | Reverse a given string            								|
